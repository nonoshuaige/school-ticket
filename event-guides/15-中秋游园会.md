# #15 中秋游园会

> 主办：校学生会 × 国学社 | 场馆：湖畔广场
> 开售：2026-09-01 10:00 | 停售：2026-09-15 18:00
> 活动：2026-09-15 18:00 - 22:00（中秋当晚）

---

## 活动简介

月上柳梢头，人约黄昏后。中秋之夜，湖畔广场变身古风游园会。

六大体验区：

| 区域 | 内容 |
|------|------|
| 灯谜长廊 | 200+ 条灯谜，猜中 3 条可兑换文创小礼品 |
| 河灯许愿 | 手写心愿放入莲花河灯，在湖畔指定区域放流 |
| 月饼工坊 | 现场 DIY 冰皮月饼（材料由主办方提供） |
| 汉服体验 | 提供 50+ 套汉服试穿，专业妆造师协助造型 |
| 手工市集 | 剪纸、面塑、糖画、中国结——非遗传承人现场教学 |
| 民乐演奏 | 古筝、琵琶、二胡、笛子，经典曲目轮番上演 |

## 注意事项

- 河灯仅限在指定区域放流，请勿在非指定水域投放
- 汉服体验区请排队等候，每人限体验 1 套
- 手工市集的非遗作品可购买，收入归非遗传承人
- 请勿攀爬湖边护栏，注意人身安全
- 请带走个人垃圾，保护镜月湖环境
- 中秋当晚气温可能转凉，建议带一件薄外套
